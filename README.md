# SignalRGB - Onikuma G67 Plugin

A custom SignalRGB plugin for the **Onikuma G67** keyboard.

- **Vendor ID:** 0x0C45
- **Product ID:** 0x8043
- **Keys:** 67 (full per-key RGB control)
- **Protocol:** Direct HID, no OpenRGB required

## Installation

1. Open SignalRGB
2. Go to **Devices** â†’ scroll down to **Third Party Services**
3. Click the **+** button to add a new addon
4. Paste this URL: `https://github.com/karim-lab/signalrgb-onikuma-g67`
5. Restart SignalRGB

## Key Map

All 67 keys are mapped with correct physical grid positions for proper effect rendering (waves, gradients, etc.).
